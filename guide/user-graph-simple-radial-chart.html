# Simple Radial Chart

# Introduction

This guide will explain how to add a simple radial chart. In fact the radial chart is just a VectorItem, and has a more simple approach then the other charttypes.

# Add and configure the radial chart

Start by making a new UI Component and give it the name 'Radial Guide':

Add a background:

Add a radial component: (Do not forget the z-offset value or else it will not be visible because the background will cover it!)

Add a property under the UI Component and call it 'Speed':

Add a script to the UI Component that reads the value of this 'Speed' property and writes it in the radial progress:

// read the component property values
// getProperty(propertyName)
var speed = getProperty('Speed')

// update graphical properties of UI items
// updateItem(itemName, itemProperty, value)
updateItem('radial_progress', 'RadialProgress', speed)

# Conclusion

Take the glasses and watch the radial chart you just created. Then adjust the speed property and watch the chart updating in real time.

Next you can add a real measurement by linking a tag to the Speed property. (Do not forget to scale between 0-100!)
