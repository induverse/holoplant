# AR Enviroment Alignment

# Introduction

The AR Environment needs to be aligned to the environment in a persistent way for two reasons:

  • Make sure the holograms are in the same place between sessions
  • Make sure that multiple users see the holograms in the same position

Holoplant realizes this alignment by the scanning if QR-based markers.

The markers only need to be scanned once. After the markers have been detected the user can freely walk around, without having to keep the QR markers in view.

# Preparation of the first two markers

In the application the QR markers where the AR environment is attached are called 'Holoplant Anchors'.

There is one special HoloplantAnchor that is automatically created and that cannot be deleted. It is called the 'OriginAnchor', and is located at the coordinates (0, 0, 0). The OriginAnchor is the only HoloplantAnchor that has no QR marker attached.

All other Anchors have a name 'WLT_x' where x is the number of the QR marker, between 1 and 999.

Create the first two HoloplantAnchors under the Default Environment: (The names can be set under the 'Edit' button)

# Printing and Physical installation of the first two markers

The QR Markers can be printed by navigating to Admin > QR Markers. Only print the markers needed, in this case #1 and #2. (If not printer is attached then print to pdf and copy to another computer before printing) Use A4 paper to print the markers. Do not trim the whitespace around the markers because without whitespace the markers will no longer be detected.

After the marker have been printed position them on the equipment or production line. The first two markers should be positioned next to each other. To allow easy scanning try to position the markers at head level. The first markers should preferably be at exact the same height. Keep a distance between the first two markers between 1 and 5 meters.

If no suiteful position can be found on the equipment or production line, then a wall nearby will also work.

# Configuring the position of the markers

No matter the size or rotation of the marker, it always only has one position and that is the left top corner where the big box is. (If the marker would be upside down then the position would still be there)

The recommended position of the first marker is (1, 0, 0). This places the marker on the X-axis, 1m away from the OriginAnchor:

The position of the second marker should be measured precisely. Since it should be on the same surface as WLT_1 then z will be 0. It should also be at the same height as WLT_1 so y will also be 0. In other words WLT_1 and WLT_2 are both on the X-axis.

Measure the distance between the top left corners of both markers precisely. (up to 1mm) Then enter the coordinate in Holoplant:

# Actual usage

After putting on the glasses and activating the Holoplant application, they will continously scan for markers. Once a marker has been detected it will display a small green indicator in the top left corner. The marker indicator will turn blue 1 second after the last scan. So colors on markers have following meanings:

  • green = currently visible to the glasses
  • blue = currently not visible to the glasses, but scanned before
  • other color or no indicator = position unknown, not scanned before

So before using the application make sure at least two markers are green or blue before proceeding

# Adding more markers

If the surface of the environment becomes big then additional markers can be added. For example every 10m.

The steps to add a new marker:

  1. Print the new marker (make sure it is a different number from all configured markers!)
  2. Physically install the marker in the desired location
  3. Scan at least two exiting markers
  4. Scan the new marker (This will add WLT_x in the list of HoloplantAnchors, with an approximate location)
  5. Optimize the position of the marker by manually measuring the position, and updating it manually in the Holoplant Application

# Conclusion

Holograms should be quite stable after this configuration. If they are shifting slightly during walking around then measure their positions again. If there are big or very big shifts happening then there are probably big mistakes in the positioning. Recommended approach is to remove all HoloPlant Anchors except WLT_1 and WLT_2, reload Holoplant app to the glasses, and position again.
