# Scripting

# getProperty(propertyName)

Gets the value of a UI Component Property. If the Property contains a structure then individual items of the structure can be accessed with 'propertyName.structureKey'.

# Usage

  • Server: no
  • Client: yes

# Arguments

  • propertyName: The name of the property from which to retrieve the value

# Returns

Bool / String / Number / Object: value of the the property. (returned datatype corresponds to the datatype of the property)

# updateItem(itemName, itemProperty, value)

Adjusts a property of one VectorItem on a UI Component.
(For example adjust the height of a polygon)

# Usage

  • Server: no
  • Client: yes

# Arguments

  • itemName: The name of the the item to adjust
  • itemProperty: The name of the property on the item to be adjusted.
  • value: The new value

# Returns

N/A

# setPropertyEndpoint(propertyName, value)

Sets to endpoint of the tag connected to the Component Property to a new value. Returns 0 if no error. Returns a non-zero value to indicate any possible errors.

# Usage

  • Server: yes
  • Client: yes

# Arguments

  • propertyName: The name of the property from which to retrieve the value
  • value: The new value of the endpoint

# Returns

Number: Error Number:

  • 0 = no error
  • 1 = property does not exist
  • 2 = property is not linked to a tag
  • 3 = property is linked to a structure and propertyName is not in the format 'propertyName.structureKey'
