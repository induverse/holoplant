# UI Component Display in AR

# Introduction

This guide will explain how to display an existing component on a panel.

# Adding a Panel

The Holoplant menu can be activated by turning your flat hand towards yourself:

Add a panel by selecting the 'Add Panel' button with your index finger:

This will add an empty panel. Try moving the panel with one or two hands to a convenient position.

# Displaying a Component on the Panel

Click the menu outside the top right corner of the panel. This will display the available components.

Select the component called 'Time and Temperature'.
(If the component is not in the list then verify the name in the component editor on your webbrowser)

You should see your component displayed:

# Make some real-time changes to the component

It is important to note that Holoplant is a real-time engineering environment. That means that most changes in the editors are immediately reflected on the AR glasses.

So a good workflow is to display the panel during the engineering next to the laptop screen to verifiy continuously if it looks correct in AR.

Position the panel next to your computer screen and open the component editor. Then make some adjustments to the component and watch the changes happen real-time on the panel.

# Conclusion

In this guide you have learned how to display a component and make your first real-time changes.

The next step is to add some scripting to the component.
