# UI Component Creation

# Introduction

This guide will explain how to design a simple UI component that will show the time and temperature in New York. The guide will only prepare the graphical components. The connection to the live data will happen in later guides.

# Create a new component

Navigate to the Components page (Side Menu > Designer > Components)

Create a new Component by clicking on the button "New Vector Component"

There should be a new component in the list with the name "new vector component". Click on the "Edit" button to open the graphical component editor:

After creating a component start with giving it a recognizable name:

A new component start with dimensions of 600 x 400. If the component is not entirely visible then the side menu can be hidden with the button in the top-left of the screen:

# Draw the background

The white rectangle indicates the size of the component. However this rectangle will not be shown on the AR glasses.

So the first step is to create a rectangle of the same size as the component:

Make sure the rectangle starts in the top left corner (with coordinates 0,0). Also ensure the width is 600 and height is 400. Make adjustments to the properties of the rectangle if necessary:

(If the properties of the rectangle are not visible then click on the rectangle)

❗️ It is recommended to assign darker colors tobackgrounds. This to avoid the rainbow effect on the Hololens and to increase eye comfort.

For example change the background rectangle to a dark blue:

The final step for the background rectangle is to give it a depth. Without a depth the background will look like a thin sheet of paper. By adding a limited depth the background will look like it's a thin plate.

Adjust the depth of the background to 10:

# Add a header bar

Add a header bar that will display the title of the panel.

First add a green rectangle and adjust following properties:

  • x = 0 and y = 0 (start at the top left corner)
  • w = 600 (same as the width of the panel)
  • h = 60 (height)
  • Fill: #37C102 (green fill)
  • z-depth = 10 (give the bar some depth)
  • z-offset = 10 (put the bar just in front of the background, this number is the same as x + z-depth = 0 + 10 of the background panel)

Then add a header text:

Set following properties on the header text:

  • x = 20 (leave some space from the left side)
  • y = 10 (leave some space from the top)
  • w = 580 (width covering the rest of the panel width)
  • h = 50 (height covering the rest of the header bar)
  • Text Color: #FFFFFF (white text color)
  • Text Size: 32
  • z-offset: 21 (let the text float just in front of the header bar. The front of the header bar is at 10 + 10 = 20)

# Add the panel content

The last step is to add the panel content. Try to make following result:

Some tips:

  • put the z-offset for the time and temperature text at 15, so it floats a little bit in front of the background
  • make the vertical bar with a height of 240, which will make it easy later when calculating the height based on the real temperature (4 x 60 = 240)
  • give the bar the same z-depth as it's width, so it will have a square shape, for example w = z-depth = 30
  • make sure everything is floating in front of the background, so at least a z-offset of 10 for items with a depth, and at least 11 for items without a depth
