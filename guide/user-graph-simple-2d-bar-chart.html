# Simple 2D Bar Chart

# Introduction

This guide will teach how to make a simple 2D bar chart.

# Introduction to Vega-Lite

Holoplant graphs and charts are configured with a subset of the Vega-Lite graphics grammar. Vega-lite is aiming to be a common graphical declaration language that can be used by different libraries, on different platforms, and in different programming languages.

To get an understanding of how vega-lite works follow the online tutorial.

# Adding the Chart

In the current version of HoloPlant the charts are entirely configured in the Script of a UI Component with the function:

renderGraph(graphName, graphSpecification, x, y, z, width, height, depth)

Where graphSpecification is a Vega-Lite specificating.

Add following code to the Script:

// parse the string data into an object
var GraphData = JSON.parse(getProperty('ChartData'))

// prepare the vega-lite specification
// see https://vega.github.io/vega-lite-api/api/ for more explanation
const graphSpecification = {
   "$schema":"https://vega.github.io/schema/vega-lite/v4.json",
   "description":"Simple Bar Chart", 
   "data":{
      "values": GraphData   // source data
   },
   "layer": [
    {
      "mark": {
        "type": "bar",      // for bar-chart
        "color": "#37c102", // try changing the color
        "depth": 40         // try changing the depth
      },
      "encoding":{
        "x":{
          "field":"Category", // adjust according source data
          "type":"nominal"
        },
        "y":{
          "field":"Value",
          "type":"quantitative" // adjust according source data
        }
      }
    }    
  ]
}

// render the graph 
// renderGraph(graphName, vegaLiteSpecification, x, y, z, width, height, depth)
renderGraph("SimpleBarGraph", graphSpecification, 100, 20, 21, 400, 240, 0)
