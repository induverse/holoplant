# UI Component Scripting

# Introduction

This guide will add some basic scripting the the 'Time and Temperature' UI component that has been created in the first guide. (If you don't have this created this then go back and do so because this guide will build on it)

Scripts make the UI components dynamic. Each UI component has exactly one script.

The scripting is a powerful subset of javascript with ECMAScript 5.1 fully supported. ECMAScript 6.0 is mostly supported with the exception of:

  • classes
  • generators
  • unicode
  • module loaders
  • weakmap
  • weakset
  • promises
  • tail calls

The guide will contain following steps:

  1. add component properties linked to tags
  2. rename UI items
  3. link property values to UI items with scriping

# Add the script

Start by adding a basic script that reads in the component property values and displays them on the text items:

// read the component property values
// getProperty(propertyName)
var time = getProperty('Time')
var temperature = getProperty('Temperature')

// update graphical properties of UI items
// updateItem(itemName, itemProperty, value)
updateItem('TimeText', 'Text', 'Time: ' + time)
updateItem('TemperatureText', 'Text', 'Temperature: ' + temperature)

Save the script and look at the result in the glasses:

The temperature should look ok. But the Time shows the json string that has been returned by the Web API from worldtimeapi.org.

Inspect the json string by going to the tag explorer and selecting 'Show Value': (tip: open a new browser window to quickly switch between the scripting view and the tag browser)

{
  "abbreviation": "EDT",
  "client_ip": "23.82.46.51",
  "datetime": "2020-08-17T20:07:00.583649-04:00",
  "day_of_week": 1,
  "day_of_year": 230,
  "dst": true,
  "dst_from": "2020-03-08T07:00:00+00:00",
  "dst_offset": 3600,
  "dst_until": "2020-11-01T06:00:00+00:00",
  "raw_offset": -18000,
  "timezone": "America/New_York",
  "unixtime": 1597709220,
  "utc_datetime": "2020-08-18T00:07:00.583649+00:00",
  "utc_offset": "-04:00",
  "week_number": 34
}

Adjust the script to parse the json object and extract the time in hours and minutes:

// read the component property values
// getProperty(propertyName)
var time = getProperty('Time')
var temperature = getProperty('Temperature')

// process values
var timeObject = JSON.parse(time) // parse the json string into an object
var datetime = new Date(timeObject.datetime);
var niceTimeString = datetime.getHours() + ':' + datetime.getMinutes()

// update graphical properties of UI items
// updateItem(itemName, itemProperty, value)
updateItem('TimeText', 'Text', 'Time: ' + niceTimeString)
updateItem('TemperatureText', 'Text', 'Temperature: ' + temperature +' deg C')

The result in the glasses should look like this:

Finally the bar height should also reflect the temperature. The total height of the bar is 240 for a range of -20 to 40 deg. That means the height of the bar can be calulated with following formula:

barHeight = (temperature + 20) * 4

Additionally the bottom of the bar should be adjusted because the UI Component has a negative Y-axis, with 0 being at the top of the component.

barBottom = 100 + (240 - barHeight)

Adjust the script accordingly:

// read the component property values
// getProperty(propertyName)
var time = getProperty('Time')
var temperature = getProperty('Temperature')

// process values
var timeObject = JSON.parse(time) // parse the json string into an object
var datetime = new Date(timeObject.datetime);
var niceTimeString = datetime.getHours() + ':' + datetime.getMinutes()
var barHeight = (temperature + 20) * 4
var barBottom = 100 + (240 - barHeight)

// update graphical properties of UI items
// updateItem(itemName, itemProperty, value)
updateItem('TimeText', 'Text', 'Time: ' + niceTimeString)
updateItem('TemperatureText', 'Text', 'Temperature: ' + temperature +' deg C')
updateItem('TemperatureBar', 'dy', barHeight) // dy controls the height property of an item
updateItem('TemperatureBar', 'y', barBottom)
